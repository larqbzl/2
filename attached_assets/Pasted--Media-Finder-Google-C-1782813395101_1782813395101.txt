«Нужно переделать поиск изображений в Media Finder. Убери (или оставь неактивным про запас) Google Custom Search — он не нужен, вместо него подключаем DataForSEO Google Images SERP API (Live mode). Делай поэтапно, тестируй сам после каждого этапа перед тем как сказать "готово".
ЭТАП 1 — Ключ и базовый поиск через DataForSEO

В Settings добавь поля DataForSEO Login и DataForSEO Password (Basic Auth). Та же архитектура хранения, что у Pexels/Pixabay — сервер, не браузер. Save/Test/Clear, статус Saved/Not set.
Серверный эндпоинт POST /api/proxy/dataforseo-images — принимает query и count (сколько картинок нужно по этому запросу), шлёт запрос на https://api.dataforseo.com/v3/serp/google/images/live/advanced с Basic Auth.
GET /api/proxy/dataforseo-images/test для кнопки Test.
Безопасный парсинг ответа (не падать на ошибках/не-JSON).
Подключи как источник на странице Search.
Протестируй через curl, покажи сырой ответ. Сделай тестовый запрос через интерфейс ("Protocol Index hoodie", count=20) — покажи, что выдача релевантная.

ЭТАП 2 — Batch с количеством картинок на запрос и параллельной обработкой (только после моего ок по этапу 1)

В Batch — таблица или список строк, где для каждой строки можно ввести: текст запроса + отдельное поле "сколько картинок" (число, например 20).
Все строки обрабатываются параллельно (несколько запросов одновременно, не по очереди один за другим), чтобы 10 запросов не ждали 60 секунд, а заняли 5-10 секунд суммарно.
Результаты группируются по блокам — один блок на одну исходную строку-запрос, с заголовком (текст запроса) и количеством найденных картинок.
Прогресс-индикатор: "X из Y запросов обработано".
Протестируй на 5 строках по 10-20 картинок каждая, покажи мне результат и время выполнения.

ЭТАП 3 — массовое скачивание (после ок по этапу 2)

Чекбоксы на каждой картинке + "Select all in block" на уровне блока-запроса.
Кнопка "Download selected" — архив zip, файлы разложены по подпапкам внутри архива (по одной на каждый запрос/блок), имя файла = источник + часть запроса + номер.
Протестируй: выбери все картинки из 3-4 блоков, скачай, открой архив, проверь структуру папок и целостность файлов.

После каждого этапа жди моего подтверждения, не делай всё сразу.»
